Nighthawk Support

Follow these steps to configure your device for optimal performance.

Follow these steps to update the APN on your Nighthawk for reliable connectivity.

  1. Download the NETGEAR Mobile App: If you haven't already, download and install the NETGEAR Mobile app.
  2. Connect to Your Router: Ensure your phone is connected to the Nighthawk's Wi-Fi network.
  3. Router Admin Login: Log in with username admin and password attadmin.
  4. Access Mobile Settings: Navigate to Settings > Advanced > Mobile > APN.
  5. Add a New APN: Tap Add to create a new profile.
  6. Enter APN Details: Type the required APN provided in your activation email and leave other fields blank.
  7. Save & Restart: Tap Save and reboot your Nighthawk router.

Manually select a network if you are experiencing signal fluctuations.

  1. Connect to Wi-Fi: Ensure your device is connected to the Nighthawk.
  2. Router Admin Login: Use admin and attadmin to sign in.
  3. Access Mobile Settings: Go to Settings > Advanced > Mobile.
  4. Network Selection: Switch from Auto to Manual.
  5. Choose Carrier: Select your preferred local carrier from the list.
  6. Apply Settings: Save changes and restart the device.

Lock specific LTE bands to prevent switching and improve stability.

  1. Connect to Router: Connect to the routers Wi-Fi network.
  2. Router Admin Login: Use admin and attadmin.
  3. Access Mobile Settings: Navigate to Settings > Advanced > Mobile.
  4. Select LTE Bands: Tap LTE Bands.
  5. Choose Preferred Band: Select the band that provides the best speeds in your area.
  6. Apply: Save the settings and reboot.
